# Neon Task Dashboard

A modern cyberpunk-inspired task management dashboard built with HTML, CSS and JavaScript.

This project was created to improve my frontend development skills through real project-based learning instead of only following tutorials.

---

## Preview

(Currently in development)

Future updates will include screenshots and live demo links.

## Features

### Current Features
- Responsive dashboard layout
- Four task columns:
  - Todo
  - In Progress
  - Review
  - Done
- Task cards
- Priority selector
- Date selector
- Live task search
- Neon/glassmorphism UI effects
- Hover animations
- CSS Grid layout
- Flexbox alignment
